This playground folder will contains one-off JavaScript files where we can explore some sort of fundamental JavaScript feature outside the context of the react application.

Files Name:                      Context of File:
> es6-array-destructuring.js     Playing around with ES6 destructuring of arrays.
> es6-object-destructuring.js    Playing around with ES6 destructuring of objects.
> hoc.js                         Playing around with React Higher Order Components.
> redux-101.js                   Playing around with Redux library.
> redux-expensify.js             Playing around with Redux using Expensify application as an example.
> promises.js                    Playing around with ES6 promises.
> firebase-101.js                Playing around with firebase realtime database and CRUD operations.
> terminalCommands.txt           Useful Terminal Commands for projects.
